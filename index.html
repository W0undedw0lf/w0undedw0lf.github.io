Dice Roller Added

Well, that was fast. With a little help from my friend we were able to get a dice roller up and running on the character sheet. We lack the means to post rolls publicly, but if you want to roll on the go and don't feel like using Avrae for some reason, it's arrived. You can specify a number of dice, the faces of the dice, and a bonus you would like to add TO EACH DICE ROLLED. This dice roller does not currently support sum total rolling unless you are rolling just one dice. It was designed with what we need it for in mind.

I'll continue to try and think up new things that might help us. Maybe just stick to some tutorial for now and see if I discover something we can use. But eventually the best improvement will come when I start learning back-end development. Which is probably a ways out. That would help us with things like character sheet storage, forums, public and shared rolls, etc. Basically anything that would hold data for another user to access. So heres to hoping that happens sooner rather than later!
